# Checkout API

A NestJS-based checkout service for processing payments and orders efficiently.

## Description

Checkout API is a robust TypeScript backend service built with [NestJS](https://github.com/nestjs/nest) framework, designed to handle payment processing and order management.

## Prerequisites

- Node.js 16+
- npm or yarn

## Installation

```bash
npm install
```

## Running the Application

```bash
# development
npm run start

# watch mode
npm run start:dev

# production
npm run start:prod
```

## Testing

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```

For detailed API documentation, refer to the [NestJS Documentation](https://docs.nestjs.com).
